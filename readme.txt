Restaurateur is mobile commerce application which can setup and run your restaurant on mobile platforms such as iOS and Android.

Used Technology:
- Titanium SDK : 3.5.0 & above
- Alloy MVC
- Ti.Map module 
- Ti.admob module
- dk.napp.social module (iOS)
- facebook module (Android)
- CodeIgniter
- Bootstrap
- JSON
